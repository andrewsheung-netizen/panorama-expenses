Monthly Email Report

Auto-sends the previous month's report on the 1st of each month when the app is opened. Uses EmailJS (free, no backend needed).
1. Sign up at emailjs.com → create an Email Service → create a Template.
2. Template body must use {{{message_html}}} (triple braces = raw HTML). Also add to_name, to_email, subject.
3. Paste your Service ID, Template ID, and Public Key above.
